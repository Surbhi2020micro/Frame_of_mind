## 👨🏻‍💻 Run Locally

### Frontend

- Clone the project

```bash
  git clone https://github.com/kasuken/Moodflix
```

- Go to the project directory

```bash
  cd Frontend
```

- Install dependencies

```bash
  npm install
```
- Start the server

```bash
  npm start
```
